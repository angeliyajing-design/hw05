# 调试记录 debug_notes

## 问题1：数据集下载超时 / 下载失败
- 现象：运行代码时卡在 MNIST 下载，出现连接超时或报错。
- 原因：网络环境限制，无法正常访问官方数据源。
- 解决：将数据集手动放入项目 `data/MNIST/raw` 目录，或使用国内镜像源加速下载。

## 问题2：matplotlib 中文显示乱码 / 方框
- 现象：绘制损失曲线时标题、标签显示为方框。
- 原因：matplotlib 默认未配置中文字体。
- 解决：在代码开头添加字体配置，指定 `SimHei` 并关闭负号修正。

## 问题3：PyCharm 提示未解析库引用（torch / matplotlib）
- 现象：代码导入包下方出现红色波浪线，提示未解析引用。
- 原因：PyCharm 未正确识别当前虚拟环境。
- 解决：重启 PyCharm，确认解释器为项目虚拟环境，重新加载依赖。

## 问题4：模型运行时报维度不匹配错误
- 现象：LeNet-5 或 CNN 模型报错张量 shape 不匹配。
- 原因：卷积、池化后特征图尺寸计算错误，全连接层输入维度不对。
- 解决：重新计算特征图尺寸，修正全连接层输入维度，确保与网络结构一致。

## 问题5：训练速度慢，未使用 GPU
- 现象：代码显示使用 cpu，训练一轮时间较长。
- 原因：未安装对应 CUDA 版本的 PyTorch，或无 NVIDIA 显卡。
- 解决：安装 GPU 版 PyTorch；无 GPU 则继续使用 CPU 训练，不影响结果。

## 问题6：运行时出现红色进度条刷屏
- 现象：训练过程中不断出现 100% 红色进度条输出。
- 原因：DataLoader 加载数据时的正常进度输出，并非报错。
- 解决：属于正常运行状态，无需处理，等待训练完成即可。
